Bayesian Confidence Scoring

Overview

This dashboard visualizes automatic confidence scoring for OpenCTI threat intelligence entities using Bayesian networks. Confidence levels are computed based on relationships and evidence propagation.

How It Works

  • Priors: Default confidence based on entity type
  • Semantic relation weights (e.g., indicates > uses)
  • Time-decay half-lives by type
  • Parent-cap (top-K) for stability & speed
  • EMA + hysteresis for output stability

Features

  • Interactive graph visualization
  • Explainable AI: contribution analysis and evidence paths
  • Confidence history tracking
  • Configurable parameters in config/bayes.yaml
